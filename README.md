# bat_box
バッチファイル置き場

## Overview
名前を書いて、下に概要を書く
*shutdown.bat
/t の後ろの秒数を変更すればシャットダウンまでの秒数が指定可能（デフォルトは60秒）
*バックアップバッチ
## Description
タスクマネージャーでbatファイルを自動起動するのがオススメ
PCの自動起動についてはBIOS側で設定する、マザーボードによっては搭載していない場合有り
